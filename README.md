Project Overview:

This project focuses on analyzing credit card transactions and customer data using Power BI. The goal is to transform raw data into meaningful insights through interactive dashboards.

The dashboard helps in understanding customer behavior, transaction trends, and spending patterns, enabling better business decisions.

Tools & Technologies Used:

Power BI
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)

Dataset Information:

The dataset used in this project includes:
Customer details
Transaction data
Card category
Transaction amount
Date of transaction
Customer demographics

Data Processing Steps:

Imported Excel dataset into Power BI
Cleaned and transformed data using Power Query
Handled missing values and formatted columns
Created calculated measures using DAX
Built relationships between tables

Key Insights:

Identified high-value customers
Analyzed spending patterns across different card categories
Observed transaction trends over time
Compared customer segments based on behavior
